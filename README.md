# Simple Django Portfolio Website

A basic portfolio website built with Django for the backend and HTML/CSS/JavaScript for the frontend. It runs locally using the Django development server.

## Run locally

1. Open a terminal in the project folder.
2. Start the Django server:

```bash
C:/Users/User/AppData/Local/Programs/Python/Python312/python.exe manage.py runserver
```

3. Open http://127.0.0.1:8000 in your browser.

## Features

- Homepage with introduction and portfolio sections
- Skills area and project cards
- Contact link via email
- Responsive styling with CSS
- Simple JavaScript for page load logging
